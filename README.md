# GENAI
College FAQs Assistant using LLMs and RAG

This project is a Domain-Specific AI Chatbot developed using Retrieval-Augmented Generation (RAG) and Large Language Models (LLMs) to provide accurate answers for college-related queries. The system is implemented in Google Colab using Python, LangChain, ChromaDB, Ollama, and Llama 3.

🚀 Features

College FAQ chatbot using RAG architecture
Semantic search with vector embeddings
Accurate and context-aware responses
Reduced hallucination using retrieved knowledge
Local Llama 3 deployment using Ollama
Interactive interface with Streamlit
PDF and TXT document support
🛠️ Technologies Used
Python
Google Colab
LangChain
Ollama
Llama 3
ChromaDB
HuggingFace Embeddings
Streamlit

⚙️ Workflow

Load college documents
Split text into chunks
Generate embeddings
Store embeddings in ChromaDB
Retrieve relevant chunks
Generate responses using Llama 3

📌 Applications

College FAQ Assistant
Student Support System
Academic Information Retrieval
AI-based Institutional Helpdesk

▶️ How to Run

Open the notebook in Google Colab
Install required libraries
Run all notebook cells sequentially
Start asking college-related questions

📖 Conclusion

The project demonstrates how RAG and LLMs can build reliable, domain-specific AI assistants with accurate and context-aware responses for educational institutions.
